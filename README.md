# Lillz Skincare - Premium Beauty Solutions

A modern, responsive skincare website built with HTML, CSS, and JavaScript.

## 🌟 Features

- **Responsive Design**: Looks great on all devices (mobile, tablet, desktop)
- **Product Showcase**: Beautiful product cards with pricing and descriptions
- **Shopping Cart**: Add products to cart with quantity tracking
- **Testimonials**: Customer reviews and ratings
- **Contact Form**: Get in touch section with form validation
- **Smooth Navigation**: Smooth scrolling between sections
- **Benefits Section**: Highlight why customers should choose Lillz

## 📁 Project Structure

```
├── index.html      # Main HTML file
├── styles.css      # All CSS styling
├── script.js       # JavaScript functionality
└── README.md       # This file
```
